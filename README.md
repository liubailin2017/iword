# iword
一个跨平台的命令行词典
## 使用
	 使用: s [OPTION]... [word]  
					--en 发英音
					--am 发美英
					-j --jushi 显示例句
					-h --help 显示帮助
					-a --about 显示关于

## 编译
	编译 run.c生成功执行文件
	编译java程序 ，可用eclipse直接导出为jar把生成的jar命名为 s.jar然后和c语言生成的可执行文件放在同一目录。
